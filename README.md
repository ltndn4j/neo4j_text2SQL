# neo4j_text2SQL

This is an example to do text2SQL using GraphRAG methodology

## Installation

### Setup postgrsql

dataset can be downloaded here : https://github.com/h8/employees-database 
1. Download the bz2 file and extract the sql file (bzip2 -d employees_data.sql.bz2)
2. Restore schema and data (ex: use pgadmin4, right click on a database and chose ```restore``` with the ```plain``` sql file )
